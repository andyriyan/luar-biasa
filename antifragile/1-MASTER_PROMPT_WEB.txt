## SISTEM NAVIGASI (PENTING)

Bottom Navigation memiliki 4 item:
1. **Beranda** (icon: home) → `index.html`
2. **Materi** (icon: menu_book) → `materi.html` ← halaman daftar materi yang benar
3. **Progress** (icon: bar_chart) → `progress.html` ← tracking progress belajar
4. **Pengaturan** (icon: settings) → `settings.html` ← tema & identitas madrasah

## SISTEM TEMA

Aplikasi memiliki 4 tema yang bisa dipilih di Settings:
- `light` — Terang (default)
- `dark` — Gelap
- `nu-maarif` — NU Ma'arif (hijau #10853F + kuning #FEEA35, sesuai logo madrasah)
- `nu-maarif-dark` — NU Malam (hijau gelap + aksen kuning)

Tema disimpan di localStorage via `Storage.saveSetting('theme', themeKey)`.

## SISTEM PROGRESS

Progress belajar disimpan otomatis di localStorage via `Storage.saveProgress(slug, persen)`.
- Saat membuka tiap konsep → progress diperbarui otomatis
- Saat selesai semua konsep → progress = 100%
- Setelah quiz → progress = 100%
- Data bisa dilihat di halaman `progress.html`

## IDENTITAS MADRASAH

Default identitas:
- Nama: **MTs Ma'arif Jumo**
- Tagline: **"Madrasahe keren, ora ninggal pesantren"**
- Logo: `assets/logo/logomts.png`
Bisa diubah melalui `settings.html` dan tersimpan di localStorage.

## KOMPONEN WAJIB YANG HARUS DIGUNAKAN

Selalu gunakan injection komponen via `data-component="..."`, JANGAN hardcode markup komponen di halaman materi:

```html
<div data-component="focus-panel"></div>   ← WAJIB di setiap halaman materi
<div data-component="bottom-nav"></div>    ← WAJIB di semua halaman
```

## ATURAN KONTEN MATERI (WAJIB DIIKUTI)

- **Matematika/Rumus**: Gunakan KaTeX. Block: `$$rumus$$`, Inline: `$rumus$`
- **Bold**: `**teks**`  |  **Italic**: `*teks*`  |  **Inline code**: `` `kode` ``
- **Gambar**: `<img src="images/nama.webp" style="width:100%; border-radius:12px; margin:16px 0;" alt="deskripsi">`
- **Format JSON**: Pastikan valid, tidak ada trailing comma
- **Konsep**: Pecah menjadi 5–10 konsep agar siswa tidak bosan

## LANGKAH UPDATE `data/materi-list.json`

Buka file tersebut dan tambahkan blok ini ke dalam array `"materi": [...]` (tambah koma di objek sebelumnya):

```json
{
  "id": "[slug-materi]",
  "slug": "[slug-materi]",
  "title": "[Judul Materi]",
  "description": "[Deskripsi Singkat]",
  "category": "math",
  "thumbnail": "",
  "totalConcepts": 5,
  "estimatedTime": "20 menit",
  "difficulty": "mudah",
  "tags": ["tag1", "tag2"],
  "published": true
}
```

Nilai `category` yang valid: `"math"`, `"science"`, `"aswaja"`, `"all"` (jangan pakai selain ini).

**Halo Bapak/Ibu Guru!** Materi Anda telah saya ubah menjadi modul interaktif. Ikuti 4 langkah berikut:

**LANGKAH 1**: Buat folder `materi/[slug-materi]/`

Selesai! Buka browser dan akses via local server (http://localhost:8080).
